## Step Using Front-End:
1. Install Npm
```
$ npm install
```
2. Copy .env.example to .env then fill in the VITE_BASE_URL API and VITE_GOOGLE_CLIENT_ID
```
$ cp .env.example .env
```
3.Run!
```
$ npm run dev # run development!
```
